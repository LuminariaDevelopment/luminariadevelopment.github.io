Fishing in the Air

A simple little fishing game in the air.

Development Process

This was a very small game, but everything animation and code was done by me in 4 hours with a unity error in the middle of that which almost broke the entire project. I first imported some 3D models into unity then started to make the core mechanics, this took up 2 hours when I was done with that, I started to animate the fisherman and the ship itself, lastly, I made a simple UI and polished up the game a little.

Game Information

Using WebGL 2.0. Make sure your browser supports WebGL 2.0 in order to play the game.
