My first unity game

My first Unity game was an interesting challenge for me because, well, it was my first time touching Unity. The gameplay involved players navigating through various obstacles to reach a portal. The development process was just following a well-made YouTube tutorial by Jonas Tyroller (https://www.youtube.com/@JonasTyroller ). The game features moving platforms, narrow paths, and long jumps. Creating this game ignited my passion for game development, and I realized that with some time and effort, I could create some great games.
