Working Piano

A working piano inside of Unity.

Project Description

I wanted to make a working piano for another project a long time ago. The original project idea failed but I felt like getting a working piano inside of unity is still pretty impressive, so I decided to move the piano to its new home: A beautiful field of flowers. There the two pianos inside the project self-playing one and a manual one, they both kind of uses the same concept. Music: Merry Go Around of Life (from: Howls Moving Castle).
